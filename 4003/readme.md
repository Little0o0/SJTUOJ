#4003getMinBottles
##Description
你有n个初始装水1L的瓶子。你可以将两个包含同样多水的瓶子合并为一个瓶子，水量为两个瓶子的和。 我希望最终剩下k个瓶子，但这可能是无法做到的。 但你还可以购进若干个初始装水1L的瓶子，这样就能通过上面的操作达到要求。 试问，最少需要购进多少个瓶子。

##Input Format
一行两个正整数n,k (n<=100000000, k<=1000, 30%数据k=1)

##Output Format
一行一个整数，表示最少需要购进多少个瓶子数

##Sample Input

	4 1

##Sample Output

	0
